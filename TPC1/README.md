## Resumo
Este trabalho apresenta a especificação de uma ontologia OWL para modelar
o conhecimento presente numa narrativa académica envolvendo estudantes,
professores, cursos, línguas e instituições.

A ontologia foi desenvolvida seguindo a metodologia apresentada na UC
RPCW, permitindo a representação formal do conhecimento e a resposta
a questões semânticas sobre a história.
